Unidad 2 - Recursividad

2.1 Definición

La recursividad es un concepto que se indica cuando un método se llama a sí mismo.
Cuando creamos un método recursivo, debemos asegurarnos de que termine, lo que implica definir una condición base para evitar llamadas infinitas.

Si no se controla adecuadamente, las llamadas recursivas pueden consumir toda la memoria y hacer que el programa se rompa.

2.2 Procedimientos Recursivos

Un procedimiento activo que se invoca a sí mismo o desde otro procedimiento activo es un procedimiento recursivo. Esta acción se denomina recursión.
Los procedimientos recursivos deben declararse con el atributo RECURSIVE y contar con un límite para evitar desbordamientos de memoria debido a llamadas infinitas.

2.3 Ejemplos de Casos Recursivos

  • Los números naturales se pueden definir de la siguiente forma: 0 es un número natural y el sucesor de un número natural es también un número natural.

  • El factorial de un número natural n es 1 si dicho número es 0, o n multiplicado por el factorial de n-1 en caso contrario.

  • La n-ésima potencia de un número x es 1 si n es igual a 0, o el producto de x por la potencia (n-1)-ésima de x cuando n es mayor que 0.

Ejemplo: Factorial de un Número

import java.util.HashMap;
import java.util.Scanner;

public class FactorialDinamico {

    // Mapa para almacenar resultados calculados
    private static HashMap memo = new HashMap<>();

    // Método recursivo con memorización
    public static int factorial(int n) {
        if (n <= 1) {  // Caso base
            return 1;
        }
        // Verificar si ya está calculado
        if (memo.containsKey(n)) {
            return memo.get(n);
        }
        // Calcular y almacenar en el mapa
        int resultado = n * factorial(n - 1);
        memo.put(n, resultado);
        return resultado;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Ingrese un número para calcular su factorial: ");
        int numero = scanner.nextInt();

        if (numero < 0) {
            System.out.println("El factorial no está definido para números negativos.");
        } else {
            int resultado = factorial(numero);
            System.out.println("El factorial de " + numero + " es: " + resultado);
        }
        scanner.close();
    }
}
